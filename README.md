# Custom Nodejs driven Blog & Website

A lightweight, high-performance, dependency-conscious website and blog. This project focuses on speed, minimal overhead, and modern web standards, migrating legacy content into a streamlined, architecture. It's the migrated version from Wordpress to a custom made Nodejs app

## 🚀 Features

- **Blazing Fast Performance:** Built entirely with vanilla JavaScript and CSS—no heavy frameworks, no jQuery, no Bootstrap.
- **Modern Backend:** Powered by Node.js, Express, and EJS templates using native ECMAScript Modules (`import/export`).
- **Robust Architecture:** Nginx reverse-proxy setup.

---

## 🛠️ Tech Stack

| Layer | Technology |
| :--- | :--- |
| **Backend** | Node.js (ES Modules), Express |
| **Templating** | EJS (Embedded JavaScript templates) |
| **Web Server** | Nginx (Reverse Proxy) |
| **Frontend** | minimal Vanilla JS, CSS |
